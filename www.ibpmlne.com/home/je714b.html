Access keys

To reach the sections whose shortcuts were defined, use the combinations of the following keys :

  • on Windows : Alt + Maj + the key or Alt + the key then ENTER
  • on Macintosh : Ctrl + Maj + the key or Pomme + the key then ENTER

The parts concerned and the associated numerical keys (in top of the keyboard) are :

  • the accessibility policy (this page) - Key 0
  • the home page - Key 1
  • the main text of the page - Key 2
  • the site map - Key 3
  • the search engine - Key 4
  • the main navigation - Key 5
  • the section "About us" - Key 6
  • the section "News" - Key 7

Accessibility rules complied with :

  • Most of checkpoints of the priority level 1.
  • Some checkpoints of the priority level 2.
  • Some checkpoints of the priority level
